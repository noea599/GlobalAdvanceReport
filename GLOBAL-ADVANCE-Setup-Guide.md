## üîÑ **PROCESSING WORKFLOW (FOR HQ/COMMS TEAM)**

### **When Request Comes In:**

**STEP 1: Download ZIP File** (30 seconds)
- Team member emails ZIP or uploads to shared drive
- Download to your computer
- Extract/unzip

**STEP 2: Review Request Info** (1 minute)
- Open `REPORT-REQUEST-INFO.txt`
- See all details at a glance:
  - Who requested it
  - What type of report
  - Time period/event
  - Audience
  - Key metrics
  - Special instructions
  - Deadline

**STEP 3: Upload to Claude AI** (2 minutes)
- Go to claude.ai (or your Claude project)
- Upload all photos from the `photos` folder
- Copy the request details

**STEP 4: Generate Report** (3 minutes)
Tell Claude:
```
Generate a [Report Type] for Global Advance from these photos.

Event/Period: [from request info]
Audience: [from request info]
Program: [from request info]

Context:
- Focus on changemakers ignited
- Emphasize spiritual + economic capital lift
- Highlight Great Commission fulfillment
- [any additional context from requester]

Key Metrics: [from request info]

Please create a professional Word document with:
- Global Advance branding and messaging
- Compelling visuals descriptions
- Transformation stories
- Kingdom impact metrics
- Call to action for [audience]

Save as .docx
```

**STEP 5: Download & Email** (2 minutes)
- Download the generated .docx report
- Email to requester:

```
Subject: Your Global Advance Impact Report - [Event/Period]

Hi [Name],

Attached is your requested [Report Type] for [Event/Period].

The report highlights:
- [Key outcome 1]
- [Key outcome 2]
- [Key outcome 3]

Perfect for sharing with [audience]. Let me know if you need 
any adjustments!

In His Service,
[Your Name]
Global Advance Communications Team
```

**Total Time: 8-10 minutes**

## üìû **SUPPORT & TROUBLESHOOTING**

### **Common Issues:**

**Problem:** "Form link doesn't work"
**Solution:** 
- Wait 5 minutes after enabling GitHub Pages
- Clear browser cache
- Try incognito/private window
- Check spelling of URL

**Problem:** "Photos won't upload"
**Solution:**
- Check file size (max 10MB each)
- Ensure files are images (JPG, PNG, etc.)
- Try different browser (Chrome recommended)
- Check internet connection

**Problem:** "ZIP file too large to email"
**Solution:**
- Upload to Google Drive/Dropbox
- Share link instead
- Or split into multiple ZIP files

**Problem:** "Can't open ZIP file"
**Solution:**
- Windows: Right-click ‚Üí "Extract All"
- Mac: Double-click to extract
- Use free tool like 7-Zip if needed

**Problem:** "Form looks broken on mobile"
**Solution:**
- Clear browser cache
- Try Chrome or Safari
- Ensure latest browser version
